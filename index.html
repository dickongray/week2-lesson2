Font-Sizing

Px, Em, and Rem

Choosing a font size can make or break the readability of your webpage. Weâ€™ll talk about some general rules of thumb to go by and utilizing a typographic scale.

  • A pixel is a fixed unit of measure, it pertains to media with display screens, such as a computer.
    A good rule of thumb is to set your body text anywhere between 16 pixels,which is the default, to 24 pixels.

  • Ems and rems are relative units for font sizing. Ems are relevent to its parent space size. For example, most browsers default to a font size of 16 pixels, which is equal to one em.

    lookout for a compounding m's, meaning nested elements font sizes can begin to multiply on top of each parent.

  • Rems, which stand for route ems, have been on the rise in web design. Instead of being relative to it's parent class or element like an em, Rems are relative to the route em, set to the html

    Rems are referencing the root element and not the parent element. This also corrects the problem of compounding ems and still provides the benefits of scalability for responsive design.
